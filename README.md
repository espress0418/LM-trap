## 런천미트트랩
### 개요
런천미트트랩은 디스코드 서버 내 스팸 메시지 도배에 대응하기 위한 봇으로, 사용자와 서버에 스팸 방지용 트랩을 설치합니다.
런천미트트랩은 누구나 설치할 수 있으며, 간단한 과정만으로 손쉽게 트랩을 설치합니다.
런천미트트랩은 일반 유저가 번거로움 없이 서버에 가입하되, 해킹으로 인해 스팸 메시지를 도배하기 전에 효과적으로 잡아낼 수 있습니다.

런천미트트랩으로 당신의 서버를 외부의 위협으로부터 보호하세요!

### 기능
디스코드 봇 '런천미트트랩'은 서버 내에 트랩 채널을 생성해 스팸 메시지 도배를 사전에 방지합니다.
사용자가 해킹을 당해 스팸 메시지를 도배하다 '런천미트트랩' 에 의해 서버에서 차단당했을 경우 계정 및 데이터 구제를 위해 복구 지침을 DM으로 전송합니다.

`/트랩 설정`
- 서버 최상단에 트랩을 설치합니다.
- 관리자를 제외한 모든 유저를 대상으로 해당 채널에 메시지를 전송할 시 1시간 동안의 메시지 삭제와 함께 **즉시 서버에서 차단됩니다.**
`/트랩 해제`
- 서버에 활성화된 트랩 기능을 모두 해제합니다.

이 외의 기능 및 명령어는 공식 페이지를 참조해주세요.
